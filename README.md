اسم المشروع:
نظام الصيدلية الرقمي (Pharmacy Management System - PMS)

وصف المشروع:
نظام الصيدلية الرقمي هو تطبيق ويب ديناميكي متكامل، صُمم لأتمتة عمليات الإدارة والمبيعات داخل الصيدليات. يهدف النظام إلى تحسين تجربة العملاء عبر واجهة تفاعلية تتيح تصفح المنتجات الطبية، إدارة سلة المشتريات، وإتمام عمليات الشراء بسهولة، مع ميزة استعراض الفواتير الخاصة بكل عميل فور إتمام الطلب.
كما يوفر النظام لوحة تحكم إدارية متقدمة تُمكّن المسؤول من الإشراف الكامل على النظام، بما في ذلك مراقبة المخزون، تحديث بيانات الأدوية، معالجة الطلبات الواردة، ومتابعة سجلات المبيعات بدقة لضمان كفاءة العمليات التشغيلية.

فكرة المشروع:
تقوم فكرة المشروع على التحول الرقمي للإدارة الصيدلانية، بهدف التخلص من التعقيدات والسلبيات المرتبطة بالإدارة اليدوية. يوفر النظام منصة إلكترونية تجمع بين سهولة التسوق للعميل وقوة الإدارة للمسؤول، مما يساهم في تقليل الأخطاء البشرية ورفع كفاءة الخدمة المقدمة للمرضى.

أبرز وظائف النظام:

استعراض المنتجات الطبية مع تفاصيلها وصورها التوضيحية.

إدارة سلة التسوق وإتمام عمليات الطلب إلكترونياً.

لوحة تحكم شاملة للمسؤول لإدارة الأدوية، المخزون، والطلبات.

طريقة التشغيل:
1-استنساخ المشروع: استخدم الأمر التالي في الـ Terminal:
git clone https://github.com/reemmerwan/Pharmacy-System.git
2-إعداد قاعدة البيانات: استورد ملف (pharmacys_db.sql) الموجود داخل مجلد (database) عبر برنامج phpMyAdmin.
3-تشغيل المشروع: تأكد من نقل مجلد المشروع إلى مسار خادم الـ (C:\xampp\htdocs)، ثم افتح المتصفح وانتقل إلى الرابط التالي:
http://localhost/Pharmacy_System/index.php
